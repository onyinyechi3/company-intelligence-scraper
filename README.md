\# Company Intelligence Scraper



A Python web scraping pipeline that extracts structured data from a multi-level website and validates the resulting dataset.



\## Features



\- Multi-level scraping (category → item → details)

\- Pagination handling

\- Structured dataset generation

\- Automated dataset validation



\## Tech Stack



Python  

Requests  

BeautifulSoup  

Pandas  



\## Run the pipeline



python run.py



This will:

1\. Scrape the website

2\. Generate a dataset

3\. Run validation checks
